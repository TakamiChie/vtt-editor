# VTT Editor

WebVTT（.vtt）形式の字幕ファイルを効率的に編集・管理するためのWebベースのエディタです。

## ユーザーガイド

このエディタは、字幕制作や文字起こしデータの整形を直感的に行えるツールです。

VTTファイル(拡張子:.vttまたは.txt)を読み込むと、画面左側にタイムスタンプ、話者名、テキストのプレビューが一覧表示されます。

クリックすることで該当する編集箇所へ即座に移動できます。

### 主な機能

隣接する字幕セグメントを結合しタイムスタンプを自動調整する「Merge with Next」機能や、正規表現に対応した文字列検索、指定した文字数を下回る行を素早く特定する「Threshold（閾値）」ハイライト機能を備えています。

また、編集した内容は一つのファイルとして保存できるほか、登場する話者ごとにファイルを分割して一括保存することも可能です。

## 開発者ガイド

本プロジェクトは React と TypeScript をベースに、ビルドツールとして Vite を採用しています。

開発環境を構築するには、まず Node.js がインストールされていることを確認し、プロジェクトのルートディレクトリで `npm install` を実行して依存パッケージをインストールしてください。

その後、`npm run dev` を実行することでローカル開発サーバーが起動します。主要なロジックは `src/VttEditor.tsx` に集約されており、字幕のパース、マージアルゴリズム、検索フィルタリング、および Blob を利用したエクスポート処理が実装されています。

UIの状態管理には React Hooks（useState, useMemo, useEffect）を活用し、ユーザー設定の永続化には Web Storage API (localStorage) を使用しています。

### コマンド

- `npm run dev`: 開発サーバーの起動
- `npm run build`: プロダクション用ビルドの生成
- `npm run preview`: ビルド後のプレビュー実行

### デプロイ

本プロジェクトは GitHub Actions を使用して GitHub Pages にデプロイするように設定されています。`main` ブランチにプッシュされると自動的にビルドが走り、サイトが更新されます。

※デプロイ前に `vite.config.ts` の `base` プロパティがリポジトリ名と一致しているか確認してください。
